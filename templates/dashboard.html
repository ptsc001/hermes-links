📱 iPhone Shortcut einrichten

1
Shortcuts-App öffnen

Automatisierung → Neuer Kurzbefehl

2
Share Sheet aktivieren

App „Kurzbefehle" → Tippe auf „i" → In Teilen-Blatt anzeigen ✅

3
Aktionen hinzufügen

• „Eingabe empfangen" von Kurzbefehlen
• „URL" – Inhalt aus Eingabe
• „Text abfragen" für Notiz
• „Inhalte der URL abrufen" – POST an API

4
API-URL konfigurieren

POST Anfrage an:

https://links.innovanext.de/links/api/links

Header: X-API-Key mit deinem Schlüssel

Body: JSON {"url": "...", "note": "..."}
